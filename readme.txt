== Installation ==
= General Requirements =
* PHP: 7.0 or newer

= PHP Notice =
* Plugin doesn't work with PHP 5.6 or older versions.

= WordPress Requirements =
* WordPress: 5.3 or newer

= WordPress Notice =
* Plugin doesn't work with WordPress 5.2 or older versions.

= Basic Installation =
* Plugin folder in the WordPress plugins should be `coreseo`.
* Upload `coreseo` folder to the `/wp-content/plugins/` directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
* Check all the plugin settings before using the plugin.
